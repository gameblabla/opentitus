===========
OpenTitus
===========

Port to GCW0 by gameblabla.
OpenTitus is an open source reimplementation of Titus the fox, a game originally made by Titus in 1992
and is in fact, a rebranded version of the game called Moktar, made by Titus as well.

===========
How to use
===========

OpenTitus requires "Titus the fox" MS-DOS port files or Moktar.
Put the folder .opentitus in /media/home/
If you have the "Titus the fox" files, put them in /media/home/.opentitus/titus
For Moktar, put them in /media/home/.opentitus/moktar

You can then start the game "OpenTitus" from Gmenu2X as usual.
